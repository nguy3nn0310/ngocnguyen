Lesson 1 / Bài 1

Hero Section / Phần mở đầu

English: A hero section introduces the page and gives visitors a clear first impression.

Tiếng Việt: Hero section giới thiệu trang và tạo ấn tượng đầu tiên rõ ràng cho người xem.

HTML

<section class="hero">
  <h1>Learn Technology</h1>
  <p>Learn • Share • Grow Together</p>
</section>
English: The section groups the content, h1 is the main title, and p adds supporting text.
Tiếng Việt: section gom nội dung lại, h1 là tiêu đề chính, còn p là dòng mô tả hỗ trợ.

Lesson 2 / Bài 2

Navigation Bar / Thanh điều hướng

English: Navigation links help visitors move between sections and pages.

Tiếng Việt: Các liên kết điều hướng giúp người xem di chuyển giữa những phần và trang khác nhau.

<nav>
  <a href="index.html">Home</a>
  <a href="learn-technology.html">Learn Technology</a>
</nav>
English: Each a element creates a clickable link.
Tiếng Việt: Mỗi thẻ a tạo một liên kết có thể bấm.
Practice / Thực hành: Add a Contact link. / Thêm liên kết Contact.

Buttons / Nút bấm

English: Buttons guide visitors toward an action.

Tiếng Việt: Nút bấm hướng người xem đến một hành động cụ thể.

<a class="btn" href="#html-course">
  Start Learning
</a>
English: This is a link styled to look like a button.
Tiếng Việt: Đây là liên kết được trang trí để trông giống nút bấm.

Tiếng Việt: Card sắp xếp thông tin liên quan thành từng khối dễ đọc.

Lesson 5 / Bài 5

Fix Uneven Cards with Flexbox / Sửa card không đều bằng Flexbox

English: This lesson comes from a real layout problem on this website: the four course cards did not look even because each card had a different amount of text.

Tiếng Việt: Bài học này xuất phát từ một lỗi bố cục thật trên website: bốn card khóa học nhìn không đều vì mỗi card có lượng nội dung khác nhau.

English: Each card grows only as much as its content needs. More text creates a taller card, and the buttons do not line up.
Tiếng Việt: Mỗi card chỉ cao theo lượng nội dung bên trong. Card có nhiều chữ sẽ cao hơn và các nút không nằm thẳng hàng.

Original CSS / CSS ban đầu

.course-card{
  padding:24px;
}

English: This code only adds inner spacing. It does not control card height or button position.

Tiếng Việt: Đoạn code này chỉ tạo khoảng cách bên trong. Nó không kiểm soát chiều cao card hoặc vị trí nút.

Solution / Cách sửa

.course-grid{
  display:grid;
  align-items:stretch;
}

.course-card{
  display:flex;
  flex-direction:column;
  min-height:300px;
}

.course-card .btn{
  margin-top:auto;
}

Line-by-Line Explanation / Giải thích từng dòng

display: grid
English: Creates the card layout.
Tiếng Việt: Tạo bố cục dạng lưới cho các card.
align-items: stretch
English: Makes cards in the same row stretch to equal height.
Tiếng Việt: Làm các card trong cùng một hàng giãn ra bằng chiều cao.
display: flex + flex-direction: column
English: Turns each card into a vertical Flexbox container.
Tiếng Việt: Biến mỗi card thành một Flexbox theo chiều dọc.
min-height: 300px
English: Gives every card a consistent minimum height.
Tiếng Việt: Tạo chiều cao tối thiểu giống nhau cho mỗi card.
margin-top: auto
English: Pushes the button to the bottom so all buttons align.
Tiếng Việt: Đẩy nút xuống cuối card để tất cả các nút nằm thẳng hàng.

Lesson 6 / Bài 6

Links / Liên kết

English: Links connect pages, files, email addresses, and outside websites.

Tiếng Việt: Liên kết kết nối trang, tệp, email và website bên ngoài.

<a href="html-course.html">
  Open HTML Course
</a>
English: href tells the browser where to go.
Tiếng Việt: href cho trình duyệt biết phải đi đến đâu.
Practice / Thực hành: Link to your Resume page. / Tạo liên kết đến trang Resume.
Lesson 7 / Bài 7

Forms / Biểu mẫu

English: Forms collect information from visitors.

Tiếng Việt: Biểu mẫu thu thập thông tin từ người xem.

<form>
  <label for="name">Name</label>
  <input id="name" name="name" type="text">
  <button type="submit">Send</button>
</form>
English: label explains the field, input collects information, and button submits it.
Tiếng Việt: label giải thích ô nhập, input nhận dữ liệu, còn button gửi biểu mẫu.
Practice / Thực hành: Add an email field. / Thêm ô nhập email.
Lesson 8 / Bài 8

Tables / Bảng

English: Tables display structured information in rows and columns.

Tiếng Việt: Bảng trình bày thông tin có cấu trúc theo hàng và cột.

<table>
  <tr><th>Lesson</th><th>Topic</th></tr>
  <tr><td>1</td><td>Hero Section</td></tr>
</table>
English: tr creates a row, th creates a heading cell, and td creates a data cell.
Tiếng Việt: tr tạo hàng, th tạo ô tiêu đề, còn td tạo ô dữ liệu.
Practice / Thực hành: Add another lesson row. / Thêm một hàng bài học mới.
Lesson 9 / Bài 9

Responsive Design / Thiết kế responsive

English: Responsive design helps a webpage work on phones, tablets, and computers.

Tiếng Việt: Thiết kế responsive giúp trang web hiển thị tốt trên điện thoại, máy tính bảng và máy tính.

<meta name="viewport"
content="width=device-width, initial-scale=1">

@media (max-width: 600px) {
  h1 { font-size: 38px; }
}
English: The viewport setting and media query adjust the page for small screens.
Tiếng Việt: Viewport và media query điều chỉnh trang cho màn hình nhỏ.
Practice / Thực hành: Reduce padding on small screens. / Giảm padding trên màn hình nhỏ.
Lesson 10 / Bài 10

Build a Homepage / Hoàn thiện trang chủ

English: Combine headings, navigation, cards, images, links, and responsive design into one page.

Tiếng Việt: Kết hợp tiêu đề, điều hướng, card, hình ảnh, liên kết và responsive thành một trang hoàn chỉnh.

<header>...</header>
<main>
  <section class="hero">...</section>
  <section class="courses">...</section>
</main>
<footer>...</footer>
English: A complete page uses clear sections and meaningful HTML elements.
Tiếng Việt: Một trang hoàn chỉnh sử dụng các phần rõ ràng và thẻ HTML có ý nghĩa.
